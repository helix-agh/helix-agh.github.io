We are happy to announce the release of **lonkit** on [PyPI](https://pypi.org/project/lonkit/)!

**lonkit** is a Python library for constructing, analyzing, and visualizing
Local Optima Networks (LONs) for both continuous and discrete optimization problems.
LONs reveal how local optima are connected and how difficult it may be to find global optima.

Install it with:

```bash
pip install lonkit
```

Key features include Basin-Hopping, Iterated Local Search sampling,
Compressed Monotonic LONs, landscape metrics, and 2D/3D visualizations.

Check out the [documentation](https://helix-agh.github.io/lonkit) and the
[GitHub repository](https://github.com/helix-agh/lonkit) to get started.
